# 🔮 塔罗牌解锁系统指南

## ✨ 功能概述

当你首次解锁一个新动物时，会弹出一张精美的塔罗牌样式祝贺卡片，每个动物只会显示一次，让解锁体验更加特殊和有仪式感。

## 🎨 塔罗牌设计特色

### 视觉效果
- **深色神秘背景**: 深棕色渐变，营造神秘氛围
- **金色边框装饰**: 华丽的金色边框和角落装饰
- **闪烁星星**: 5颗星星在卡片周围闪烁
- **3D翻转动画**: 卡片从侧面翻转出现
- **发光效果**: 动物表情符号带有金色光晕

### 卡片结构
1. **标题区域**: 英文塔罗牌名称 + 中文副标题
2. **动物区域**: 大号动物表情符号 + 动物名称 + 描述
3. **祝福区域**: 寓意深刻的祝福语和人生感悟
4. **交互按钮**: "收下祝福"按钮

## 🎯 触发机制

### 首次解锁
- 当动物首次被解锁时，自动显示塔罗牌
- 塔罗牌显示后，该动物会被标记为"已显示"
- 后续再次解锁同一动物时，只显示简单的成功提示

### 显示逻辑
1. 检查动物是否首次解锁
2. 检查是否已显示过该动物的塔罗牌
3. 如果是首次且未显示过，则显示塔罗牌
4. 显示后标记为已显示，避免重复

## 🎮 交互方式

### 关闭塔罗牌
- **点击"收下祝福"按钮**: 主要关闭方式
- **点击背景区域**: 快速关闭
- **自动关闭**: 8秒后自动关闭
- **ESC键**: 键盘快捷键关闭

### 动画效果
- **出现**: 3D翻转 + 缩放动画 (2秒)
- **消失**: 反向翻转 + 淡出动画 (0.8秒)
- **星星闪烁**: 持续闪烁动画
- **动物发光**: 呼吸式发光效果

## 💾 数据存储

### 本地存储
- `shownTarotCards`: 存储已显示过塔罗牌的动物列表
- 与 `unlockedAnimals` 分开存储，独立管理

### 重置方法
```javascript
// 重置所有塔罗牌显示状态
localStorage.removeItem('shownTarotCards');

// 重置特定动物的塔罗牌状态
const shownCards = JSON.parse(localStorage.getItem('shownTarotCards') || '[]');
const filteredCards = shownCards.filter(card => card !== 'animalType');
localStorage.setItem('shownTarotCards', JSON.stringify(filteredCards));
```
