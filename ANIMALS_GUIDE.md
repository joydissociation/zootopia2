# 🦌🦊🦜🐧🦫🦥 动物解锁指南

## 🎯 新增动物种类

### 1. 【身体健康】- 🦌 小鹿 (Wellness Deer)
- **代表**: 身体照顾、运动、饮食、睡眠
- **解锁关键词**: 身体、健康、运动、饮食、睡眠、喝水、锻炼、瑜伽、散步、伸展
- **性格**: 温柔、安静、优雅
- **对话风格**: "记得多喝水哦～" "今天的身体感觉怎么样？"

### 2. 【情绪关怀】- 🦊 小狐狸 (Emotion Fox)
- **代表**: 情绪表达、心理健康、压力管理
- **解锁关键词**: 情绪、心情、压力、焦虑、难过、开心、冥想、深呼吸、倾诉、心理
- **性格**: 敏感、善解人意、聪慧
- **对话风格**: "今天感觉怎么样？" "所有情绪都值得被看见哦"

### 3. 【创造力】- 🦜 鹦鹉 (Creative Parrot)
- **代表**: 创作、学习、兴趣爱好
- **解锁关键词**: 创作、学习、画画、写作、读书、音乐、技能、兴趣、创意、艺术
- **性格**: 活泼、好奇、爱表达
- **对话风格**: "今天想创作什么？" "我学会了一句新话！"

### 4. 【社交连接】- 🐧 企鹅 (Social Penguin)
- **代表**: 人际关系、社交互动
- **解锁关键词**: 社交、朋友、家人、聊天、交流、帮助、感谢、人际关系、陪伴
- **性格**: 友善、温暖、群居
- **对话风格**: "今天有想念的人吗？" "记得你不是一个人哦"

### 5. 【生活整理】- 🦫 海狸 (Organized Beaver)
- **代表**: 空间整理、生活规划、环境优化
- **解锁关键词**: 整理、收纳、规划、清洁、断舍离、环境、家务、条理、计划
- **性格**: 勤劳、有条理、踏实
- **对话风格**: "今天要整理什么呢？" "一点点来，不着急"

### 6. 【休息放松】- 🦥 树懒 (Rest Sloth)
- **代表**: 休息、放松、什么都不做
- **解锁关键词**: 休息、放松、睡觉、发呆、泡澡、游戏、娱乐、慢节奏、放假
- **性格**: 慵懒、治愈、佛系
- **对话风格**: "累了就休息吧～" "什么都不做也很棒哦"

## 🔓 如何解锁动物

### 方法1: 通用对话框
在首页的"和动物朋友们聊聊"对话框中输入包含关键词的消息，例如：
- "今天我想要运动一下" → 解锁小鹿 🦌
- "我感觉有点焦虑" → 解锁小狐狸 🦊
- "想要画画创作" → 解锁鹦鹉 🦜
- "想念朋友了" → 解锁企鹅 🐧
- "房间需要整理" → 解锁海狸 🦫
- "好累想休息" → 解锁树懒 🦥

### 方法2: 创建相关任务
创建任务时选择对应的园区也可能触发动物解锁。

## 🎮 功能特点

### 首页显示
- ✅ 已解锁动物：显示完整信息和表情
- 🔒 未解锁动物：显示灰色问号，提示"完成相关任务解锁"

### 动物图鉴
- ✅ 已解锁：显示完整动物信息
- 🔒 未解锁：显示灰色格子和问号

### 聊天功能
- 只有已解锁的动物才会出现在聊天选择列表中
- 每个动物都有独特的对话风格和性格

### 解锁动画
- 解锁新动物时会显示特殊的通知动画
- 动物卡片会有闪光效果

## 🧪 测试功能

在 `test-animals.html` 页面中提供了快速测试按钮：
- **解锁所有动物**: 一键解锁所有动物用于测试
- **重置动物**: 重置为只有小猫咪的初始状态
- **显示已解锁**: 在控制台显示当前已解锁的动物列表

## 💾 数据存储

- 本地存储：解锁状态保存在 localStorage 中
- 数据库存储：如果配置了Supabase，解锁的动物会自动创建数据库记录
- 跨设备同步：通过Supabase实现多设备数据同步

## 🎯 使用建议

1. **循序渐进**: 从小猫咪开始，逐步解锁其他动物
2. **关键词触发**: 在对话中自然地提到相关话题
3. **多样化体验**: 尝试与不同性格的动物聊天
4. **任务创建**: 为不同园区创建任务来丰富体验

享受与你的动物朋友们的互动吧！🌟
