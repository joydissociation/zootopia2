# 🎲 动物任务系统功能指南

## ✨ 新功能概述

虚拟动物园应用现在支持**动物任务系统**！用户可以点击动物卡片查看任务列表，使用骰子按钮随机生成self care任务，完成任务获得经验值帮助动物成长。

## 🎯 核心功能

### 1. 点击动物卡片
- **操作**: 点击任意已解锁的动物卡片
- **效果**: 打开该动物的专属任务列表界面
- **显示**: 动物名称、当前任务、完成状态

### 2. 🎲 骰子生成任务
- **位置**: 任务列表界面顶部的金色骰子按钮
- **功能**: 随机生成该动物类型的self care任务
- **动画**: 1秒的骰子滚动动画效果
- **奖励**: 随机生成的任务给予15经验值（比手动创建的多5点）

### 3. ✅ 任务管理
- **完成任务**: 点击"✓ 完成"按钮，获得经验值奖励
- **删除任务**: 点击"🗑️ 删除"按钮，移除不需要的任务
- **任务状态**: 已完成的任务显示删除线和灰色样式

## 🎨 界面设计特色

### 骰子按钮设计
- **视觉**: 金色渐变背景，立体阴影效果
- **动画**: 悬停时的光泽扫过效果
- **滚动**: 点击时1秒的3D旋转动画
- **反馈**: 生成成功后的消息提示

### 任务卡片设计
- **布局**: 清晰的标题、园区标签、经验值显示
- **状态**: 未完成/已完成的视觉区分
- **操作**: 完成和删除按钮的像素风格设计
- **动画**: 悬停时的轻微上浮效果

### 主题适配
- **可爱梦幻**: 柔和色彩，圆角设计
- **复古波普**: 鲜艳色彩，方角设计，立体阴影
- **动物森友会**: 自然色彩，简洁边框

## 🔧 技术实现

### 数据存储
- **Supabase**: 任务存储在tasks表中
- **本地存储**: 离线模式下使用localStorage
- **关联**: 任务与动物ID关联，支持经验值更新

### 任务生成算法
```javascript
// 随机选择任务
const tasks = SELF_CARE_TASKS[gardenZone];
const randomIndex = Math.floor(Math.random() * tasks.length);
return tasks[randomIndex];
```

### 经验值系统
- **手动创建任务**: 10经验值
- **骰子生成任务**: 15经验值
- **完成任务**: 自动增加动物经验值
- **成长系统**: 经验值影响动物成长阶段

## 🚀 使用流程

### 基本流程
1. **选择动物**: 点击动物卡片打开任务界面
2. **生成任务**: 点击骰子按钮生成随机任务
3. **执行任务**: 在现实中完成self care活动
4. **标记完成**: 点击完成按钮获得经验值
5. **动物成长**: 经验值积累帮助动物升级

### 高级功能
- **任务管理**: 删除不合适的任务
- **批量操作**: 一次性查看多个任务
- **进度追踪**: 通过成长进度页面查看整体进展
- **个性化**: 不同动物类型提供不同类型的任务
